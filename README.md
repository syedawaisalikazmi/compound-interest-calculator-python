# 💰 Compound Interest Calculator (Python)

This is a simple beginner Python project that calculates compound interest based on user input.

---

## 📌 Formula Used

A = P (1 + r/100) ^ t

Where:
- P = Principal amount (initial money)
- r = Interest rate (%)
- t = Time in years
- A = Final amount

---

## 🚀 Features
- Takes user input safely
- Prevents negative values
- Handles invalid input (letters, symbols)
- Clean output formatting
